# DSpico firmware patcher

https://mighty-jun.github.io/DSpico-firmware-patcher/

<br>

This tool patches a precompiled firmware with the WRFU Tester rom to enable WRFUxxed support.
<br>
The firmware binary used by this tool can be found here: https://github.com/mighty-Jun/DSpico-auto-firmware-builder
